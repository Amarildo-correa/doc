`plugins/promptdown-helpers/skills/format-prompt-card/references/style-guide.md`
é a documentação de apoio da skill — descreve o padrão visual e de dados
esperado para um card de prompt neste projeto.

## Propósito

Arquivo de referência carregado pelo agente quando a skill `format-prompt-card`
é acionada — fornece o contrato de dados e as regras de formatação sem precisar
que o agente consulte o código-fonte diretamente.
